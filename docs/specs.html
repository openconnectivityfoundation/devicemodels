An overview of all OCF specifications can be found at : https://openconnectivity.org/developer/specifications.

All OCF defined Devices can be found in the Smart Home Device specification. Note that this specification also includes a list of standardized enumeration values. The OCF Device definitions can contain a list of mandatory Resources. However it is allowed to add more Resources to a specific Device Type, so that a more functional product can be created. The adding of additional resources is up to the manufacturer of the OCF Device.

All OCF defined Resources can be found in the Resource Type specification.
